# Allen & Heath dLive TCP module

Controls the Allen & Heath dLive via MIDI or TCP commands.

## Functions:


* Mute Channel
* Mute FX Buses
* Mute Group
* Mute DCA
* Mute Assign
* DCA Assign
* Recall Scene
* Talkback On (TCP mode only)
* Virtual SoundCheck mode (TCP mode only)
